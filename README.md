# dbt Project - Learning Purpose CI/CD in GCP

This dbt project is designed for **educational purposes** to learn and understand:

- **Continuous Integration/Continuous Deployment (CI/CD)** workflows
- **Google Cloud Platform (GCP)** integration and services
- **dbt (data build tool)** as the primary data transformation tool

## Purpose

This project serves as a hands-on learning repository to explore how dbt can be integrated into CI/CD pipelines hosted on Google Cloud Platform, demonstrating best practices for automated data pipeline testing, validation, and deployment.

## Key Focus Areas

- dbt project structure and organization
- Data transformation workflows
- Automated testing and validation
- GCP service integration
- Pipeline orchestration and deployment automation
